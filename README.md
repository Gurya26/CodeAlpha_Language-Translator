# 🌍 AI Language Translator (CodeAlpha Internship)

This project is a web-based language translator built using Flask and Google Translate API.

## 🚀 Features

* Translate text into multiple languages
* Auto language detection
* Clean and modern UI
* Copy translated text

## 🛠️ Technologies Used

* Python
* Flask
* Deep Translator API
* HTML, CSS

## 🔗 Live Demo

https://codealpha-language-translator-4.onrender.com/

## 📌 How it Works

* User enters text
* Selects target language
* API processes translation
* Result displayed instantly

## 📁 Project Structure

* app.py → Backend logic
* templates/ → Frontend HTML

## ▶️ Run Locally

1. Clone the repository
git clone https://github.com/Gurya26/CodeAlpha_Language-Translator

2. Install dependencies
pip install -r requirements.txt

3. Run the app
python app.py

4. Open in browser
http://127.0.0.1:5000/
